# 项目名称



## Marvell 企业交换机底层驱动开发

基于 Marvell ARM 架构处理器平台，负责二层及弱三层交换机产品的底层驱动开发与硬件适配工作。主要完成 MAC/PHY 驱动调试、MDIO 接口操作、端口初始化、速率协商、流控配置及硬件复位逻辑实现，同时负责设备树配置、GPIO 与 LED 状态指示驱动、网口上下电及电源管理等模块开发。在开发过程中定位并解决端口协商异常、链路不稳定、驱动 Oops 及内存访问异常等底层问题，优化中断处理流程与硬件稳定性，保障交换机在满负载场景下长期可靠运行，支撑产品顺利量产交付。

基于 Marvell ARM 处理器平台，负责公司系列化二层及弱三层交换机产品的底层驱动开发与平台化适配工作。针对数十款不同端口规格、硬件配置的交换机设备，完成 MAC/PHY 驱动调试、MDIO 总线操作、端口初始化、速率协商、流控配置及硬件复位逻辑开发，同时实现设备树配置、GPIO 控制、LED 状态指示、网口上下电及硬件异常检测等模块。在多款设备适配过程中，沉淀通用驱动框架与标准化调试流程，快速定位并解决端口协商异常、链路震荡、驱动 Oops、内存访问异常等底层问题，优化中断处理与系统稳定性，保障多款设备一次性通过量产测试并顺利交付。

基于 Marvell ARM 架构处理器平台，负责二层及弱三层交换机产品的底层驱动开发与硬件适配工作，独立完成 MAC/PHY 驱动调试、MDIO 总线操作、端口状态管理、速率协商、流控配置及硬件复位逻辑实现，同时负责设备树配置、GPIO/LED 驱动、网口上下电、硬件异常检测等模块开发。针对数十款不同硬件配置的设备进行快速适配与移植，提炼并维护通用驱动底层框架，提升新机型 bringup 效率与代码复用率。在项目中独立定位并解决端口协商失败、链路震荡、PHY 访问异常、驱动 Oops、内存越界及高负载下稳定性问题，优化中断处理与硬件交互逻辑，保障设备在长期满负载运行下无异常，多款产品顺利通过量产测试并批量交付。
